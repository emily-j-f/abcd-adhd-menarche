# abcd-adhd-menarche
This is the R code used for a study on how menarche status might be associated with ADHD traits, along with moderating effects of menarche status on mental health outcomes dependent on ADHD diagnostic status. Adolescent Brain and Cognitive Development (ABCD) Study data (release 5.1) were used for all analyses.

1) Start here with the [data processing folder](https://github.com/emily-j-f/abcd-adhd-menarche/tree/main/1.%20ABCD%20data%20processing), with data processing and imputation broken down into 4 steps.
2) There is code for descriptive analyses here in the [descriptives folder](https://github.com/emily-j-f/abcd-adhd-menarche/tree/main/2.%20Descriptives).
3) Then you can run the main analyses. There are cross-sectional and longitudinal Bayesian generalised linear mixed models in the [analyses folder](https://github.com/emily-j-f/abcd-adhd-menarche/tree/main/3.%20Analyses%20(Bayesian%20GLMM)). PDFs of outputs for all models can also be fround in the analyses folder.
4) Finally, to test model robustness, you can run the models in the [sensitivity analyses folder](https://github.com/emily-j-f/abcd-adhd-menarche/tree/main/4.%20Sensitivity%20analyses)
